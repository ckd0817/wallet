# Run and deploy SmartWallet

This contains everything you need to run the app locally.

The app keeps local spending insights in the analysis tab and can use an Android screenshot session to turn payment-result pages into draft expense records.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Install Android requirements if you want to use screenshot auto-bookkeeping
3. Run the app:
   `npm run dev`
